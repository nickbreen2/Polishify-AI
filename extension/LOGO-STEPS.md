# After updating the extension logo – what to do

1. **Rebuild the extension** (from project root):
   ```bash
   npm run build:ext
   ```

2. **Reload the extension in Chrome**
   - Open `chrome://extensions`
   - Find **Polishify AI** and click the **reload** (circular arrow) button

3. **If the toolbar icon still shows the old logo**
   - Browsers cache the extension icon. On `chrome://extensions`, click **Remove** on Polishify AI, then **Load unpacked** and select:
   - `extension/output/chrome` (path relative to your project root)

4. **Where each logo is used**
   - **Toolbar (pinned icon):** Icon only – from `extension/public/icon/` (generated by `npm run generate-ext-icons` from `Logo (1).png`).
   - **Website:** Full logo – `public/full-logo.svg` (copy from `Polishify Full Logo.svg`).
   - **Extension popup & popover:** Full logo – `extension/public/full-logo.svg`.

**Changing logos later:**  
- **Icon only (toolbar):** Update `Logo (1).png` in `public/`, run `npm run generate-ext-icons`.  
- **Full logo (website, popup, popover):** Replace `public/Polishify Full Logo.svg`, then copy it to `public/full-logo.svg` and `extension/public/full-logo.svg`.
